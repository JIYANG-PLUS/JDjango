Django整型字段

所有整型字段类型：

字段类型 字段说明 值范围 对应表单组件
BinaryField 字节型字段 bytes、bytearray、memoryview
SmallIntegerField 16位整型字段 -32768~32767
PositiveSmallIntegerField 16位正整型字段 0~32767
IntegerField 32位整型字段 -2147483648~2147483647 NumberInput
PositiveIntegerField 32位正整型字段 0~2147483647
BigIntegerField 64位整型字段 -9223372036854775808~9223372036854775807 TextInput
AutoField 32位自增型字段 1~2147483647
BigAutoField 64位自增型字段 1~9223372036854775807

字段特有参数：

字段类型 特有参数 参数说明 参数取值类型
BinaryField max_length 长度上限 正整型

注意点：

  1. 若未显示设置主键字段，Django会默认加上 id = models.AutoField(primary_key=True) 这一字段。
  2. BinaryField的editable参数默认是False。Django2.1版本之前不允许设置editable参数。
