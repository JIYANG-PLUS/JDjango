Django浮点型字段

字段类型 字段说明 值范围 对应表单组件
FloatField 浮点型字段 float NumberInput
DecimalField 高精度浮点型字段 decimal NumberInput

字段特有参数：

字段类型 特有参数 参数说明 参数取值类型
DecimalField max_digits 整数位数和小数位数之和（不包括小数点） 正整型
DecimalField decimal_places 小数位数（不包括小数点） 正整型

注意点：

  1. DecimalField字段对应Python中的Decimal模块，其具备超高精度的浮点数运算，可进行金融类计算。
  2. max_digits和decimal_places两个参数必填，且max_digits的值必须大于等于decimal_places的值。
  3. 整数位数 = max_digits - decimal_places。
