Django文件类型字段

字段类型 字段说明 值范围 对应表单组件
FileField 文件上传字段 ClearableFileInput
ImageField 图片上传字段 ClearableFileInput
FilePathField 文件路径上传字段

字段特有参数（这里并未写全，用到补充）：

字段类型 特有参数 参数说明 参数取值类型
FileField upload_to 上传路径 字符型路径
FileField max_length 最大路径长度 正整数（默认100）

注意点：

  1. 若要使用FileField字段，必须在settings中设置MEDIA_ROOT基础路径。
  2. FileField不允许设置为主键字段。
  3. FileField的upload_to参数常用赋值示例：upload_to='uploads/'、upload_to='uploads/%Y/%m/%d/'
