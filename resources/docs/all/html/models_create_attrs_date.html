Django日期型字段

字段类型 字段说明 值范围 对应表单组件
DateField 日期型字段 datetime.date TextInput
DateTimeField 长日期字段 datetime.datetime TextInput
DurationField 时间戳字段 timedelta DurationField
TimeField 时间字段 datetime.time TextInput

字段特有参数：

字段类型 特有参数 参数说明 参数取值类型
DateField auto_now 仅在调用save()方法时自动更新时间 True/False
DateField auto_now_add 仅第一次创建时赋值 True/False
DateTimeField auto_now 仅在调用save()方法时自动更新时间 True/False
DateTimeField auto_now_add 仅第一次创建时赋值 True/False
TimeField auto_now 仅在调用save()方法时自动更新时间 True/False
TimeField auto_now_add 仅第一次创建时赋值 True/False

注意点：

  1. DateField和DateTimeField的区别就是：前者在Python中的类型为datetime.date，后者为datetime.datetime。
  2. DateField设置默认值时，赋值为default=date.today，取自datetime.date.today()
  3. DateTimeField设置默认值时，赋值为default=timezone.now，取自django.utils.timezone.now()
  4. 当auto_now_add=True和default=timezone.now或default=date.today同时存在时，仅auto_now_add=True有效。
  5. auto_now_add、auto_now、default三者互相排斥，同一日期字段只能存在其中一个。
  6. 当设置auto_now_add=True或auto_now=True时，默认同时设置editable=False和blank=True。
  7. 当需要具体到时分秒时，推荐使用DateTimeField。
  8. 在除了PostgreSQL之外的所有数据库类型中，DurationField尽量不要和DateTimeField作比较。
