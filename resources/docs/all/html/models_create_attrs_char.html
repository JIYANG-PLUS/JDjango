Django字符型字段

字段类型 字段说明 值范围 对应表单组件
CharField 字符型字段 TextInput
TextField 大文本字段 Textarea
EmailField 邮箱字段 TextInput
GenericIPAddressField IPv4IPv6字段 ip TextInput
SlugField 只包含字母、数字、下划线或连字符
URLField 路由字段 URL TextInput
UUIDField uuid字段 uuid

字段特有参数：（无默认值必填）

字段类型 特有参数 参数说明 参数取值类型
CharField max_length 长度上限 正整型
EmailField max_length 长度上限 正整型（默认254）
SlugField max_length 长度上限 正整型（默认50）
URLField max_length 长度上限 正整型（默认200）

注意点：

  1. 当字符串长度过长时，推荐使用TextField。
  2. EmailField会在表单中自动触发邮箱类型校验。
  3. UUIDField字段，默认值设置default=uuid.uuid4。
  4. UUIDField可以用于设置主键唯一字段，因为uuid肯定不会重复，但是作为主键的AutoField可能更加适合。UUIDField需要手动设置默认值，并不会自动生成。
